# Selenium WebDriver Framework

## Overview

This project is a Selenium WebDriver automation framework developed using Java, TestNG, and Maven. It automates common web application scenarios and follows a structured approach for writing and executing test cases.

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven

## Features

- Browser Launch
- Login Automation
- Search Functionality
- Form Submission
- Dropdown Handling
- File Upload
- Assertions
- TestNG Test Execution
- Maven Build Management

## Project Structure

- Test Classes
- Page Objects
- Utility Classes
- TestNG XML
- Maven Configuration

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA or Eclipse.
3. Install Maven dependencies.
4. Execute the following command:

```bash
mvn clean test
```

## Purpose

This is a personal learning project created to improve Selenium automation testing skills using Java, TestNG, and Maven.
